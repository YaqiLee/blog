伪选择器

:defined

::marker

css 伪元素，只能在被定义为 display: list-item; 的元素上使用。
在 marker 里面可以使用的属性有：

  1. 字体属性
  2. color
  3. text-combine-upright
  4. content

:any-link

匹配带有 herf 属性的元素

:out-of-range / :in-range

表示 input 元素的值在 min 和 max 限定之外

:empty

选择器指定的元素不包含子元素时生效。子元素只可以是元素节点或文本（包括空格）。注释或处理指令都不会产生影响。

:default

表单控件的默认值

css

cursor

给鼠标指针设置图片样式

clip-path

通过裁剪创建元素的可显示区域，区域内的显示，区域外的隐藏。
可以通过该属性将元素剪切成各种形状 clip-path MDN

shape-outside

定义浮动元素显示的形状，显示成文字包围的各种形状

shape-image-threshold

通过设定一个 alpha 通道的界限值来提取 shape-outside 值为图像的形状。

background-blend-mode

背景混合模式 MDN blend-mode

multiply - 正片叠底

darken - 变暗

overlay - 叠加

示例

mask-image

图像遮罩层,浏览器兼容性不太好，目前还在实验阶段。2021 年 6 月 21 日

filter

滤镜，图片处理 mdn 参考

aspect-ratio

盒子元素的宽高比例，Safari 不支持

backdrop-filter

背景滤镜，给元素背景添加滤镜效果

backface-visibility

此属性与 3dtransform关联使用，指定元素背对着观察者时是否可见

writing-mode

文本水平或者垂直排布以及在块级元素中文本的方向

scroll-behavior

滚动条的滚动行为,设置为 smooth 时，滚动的内容会有一个动画过渡的效果；

利用 scroll-behavior scroll-snap-type scroll-snap-align 属性实现类似于轮播图的切换效果

scrollbar

修改滚动条的样式，仅支持 webkit 的浏览器

  • ::-webkit-scrollbar — 整个滚动条.
  • ::-webkit-scrollbar-button — 滚动条上的按钮 (上下箭头).
  • ::-webkit-scrollbar-thumb — 滚动条上的滚动滑块.
  • ::-webkit-scrollbar-track — 滚动条轨道.
  • ::-webkit-scrollbar-track-piece — 滚动条没有滑块的轨道部分.
  • ::-webkit-scrollbar-corner — 当同时有垂直滚动条和水平滚动条时交汇的部分.
  • ::-webkit-resizer — 某些元素的 corner 部分的部分样式(例:textarea 的可拖动按钮).

overscroll-behavior

控制滚动条滚动到边界时的滚动行为。正常情况下，弹窗滚动条滚动到边界时
会带动弹窗下面内容的滚动

inset

inset是 top,right,bottom,left 的简写

pointer-events

鼠标事件的目标元素事件处理

示例的代码，鼠标分别放到两个元素上试一试

pointer-events: initial
pointer-events: none

border-radius

设置元素的外边框圆角，该属性是一个简写。MDN

  • border-top-left-radius
  • border-top-right-radius
  • border-bottom-right-radius
  • border-bottom-left-radius

圆角效果受 background-clip 影响，当 border-collapse 的值为 collapse 时，border-radius 属性不会被应用到表格元素上。

contain

contain mdn

IE 和 safari 不支持

font-family

z-index

box-decoration-break

属性用来定义当元素跨多行、多列或多页时，元素的片段应如何呈现

  1. 切成片
  1. 复制样式，框装饰适用于元素的每个片段，就像片段是单个元素一样。
